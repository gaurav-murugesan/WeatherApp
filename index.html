About WeatherApp

Weatherify is a simple web application that allows users to search for current weather information for any city around the world. It provides details such as temperature, humidity, wind speed, and atmospheric pressure.

This application fetches real-time weather data from a public API.

To view the weather details, search the city name in search bar and click the city from the results shown.
